## Manual Logo Update

Use Xcode to set icon assets directly:

1. Open `/Users/serkan/Documents/ofchours/OfcHoursApp/OfcHoursApp.xcodeproj`.
2. Go to `OfcHoursApp/Assets.xcassets > AppIcon`.
3. Drag your `1024x1024` PNG into the `Any Appearance` slot.
4. Optional: add `Dark` and `Tinted` variants for iOS 18+ adaptive styling.
5. Run one archive build to validate icon packaging for App Store upload.
